Politique de Confidentialité de Quraan Pulaar

Dernière mise à jour : 14 Mars 2025

Introduction

Cette politique de confidentialité décrit comment Quraan Pulaar ("nous", "notre", "nos") collecte, utilise et protège vos informations lorsque vous utilisez notre application mobile.

Collecte et utilisation des informations

Notre application collecte les informations suivantes :

  • Données d'utilisation : Nous collectons des données anonymes sur l'utilisation de l'application pour améliorer nos services.
  • Accès au microphone : L'application demande l'accès au microphone uniquement pour les fonctionnalités audio spécifiques.
  • Stockage : Nous stockons localement vos préférences et paramètres sur votre appareil.
  • Données publicitaires : Notre application utilise Google AdMob pour afficher des publicités. AdMob peut collecter et traiter certaines données comme :
    • Identifiants publicitaires de l'appareil
    • Données de localisation approximative
    • Informations techniques de l'appareil

Publicités

Notre application affiche des publicités via Google AdMob. Nous avons mis en place les mesures suivantes :

  • Aucune publicité n'est affichée pendant la lecture du contenu sacré
  • Intervalle minimum de 3 minutes entre les publicités interstitielles
  • Filtrage strict du contenu publicitaire
  • Pas de publicités personnalisées pour les utilisateurs mineurs
  • Respect des directives de Google AdMob

Pour plus d'informations sur les pratiques de confidentialité de Google AdMob, consultez : Politique de confidentialité de Google

Protection des données

Nous prenons la protection de vos données au sérieux. Toutes les données sont stockées localement sur votre appareil et ne sont pas partagées avec des tiers, à l'exception des données nécessaires pour le fonctionnement des publicités via AdMob.

Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger vos données personnelles
  • Supprimer vos données personnelles
  • Désactiver les publicités personnalisées via les paramètres de votre appareil

Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous :

  • Email : kawkumputer@gmail.com
  • Site web : https://kawkumputer.github.io/quraan-pulaar/

Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les utilisateurs seront informés des changements via les mises à jour de l'application.

Consentement

En utilisant Quraan Pulaar, vous consentez à cette politique de confidentialité et aux pratiques de traitement des données qui y sont décrites.
